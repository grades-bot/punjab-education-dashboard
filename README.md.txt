# Punjab School Education Department Dashboard

## Overview
This is an interactive analytical dashboard visualizing key metrics for the School Education Department, Government of Punjab (Pakistan).

## Technologies Used
- **HTML5/CSS3:** Grid layout and Flexbox for responsiveness.
- **JavaScript:** For count-up animations and logic.
- **Chart.js:** For rendering the Doughnut, Pie, and Bar charts.
- **FontAwesome:** For UI icons.

## Data Sources
- Total Operational Schools: 42,873
- Total Students: 10.5 Million+
- Total Teachers: 320,713
